# AI Compliance Sentinel

Continuous governance and compliance monitoring for AI systems.

## What it is
AI Compliance Sentinel provides real-time visibility into how AI systems
behave in production, detecting policy drift, compliance risks,
and governance gaps before audits or incidents occur.
